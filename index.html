📖 왕초보 사용 설명서 (꼭 읽으세요!)
1. 준비 (사진 찍기)
물고기를 눕히고 바로 옆에 신용카드를 가로로 나란히 놓으세요. 카메라를 물고기 바로 위에서 수직으로 똑바로 들고 사진을 찍으세요.
2. 사진 불러오기
아래 노란색 [내 사진 불러오기] 버튼을 눌러서 방금 찍은 사진을 선택하세요.
3. 카드 위치 알려주기 (터치)
화면에 나온 사진 속 신용카드의 왼쪽 모서리를 손가락으로 한 번 터치(클릭)하고, 이어서 오른쪽 모서리를 한 번 더 터치하세요. (노란 선이 생깁니다)
4. 물고기 길이 재기 (터치)
카드를 다 찍었으면 바로 물고기 입술 끝을 손가락으로 한 번 터치하고, 마지막으로 꼬리 끝을 터치하세요. (파란 선이 생깁니다)
5. 서열 확인
성함을 적고 파란색 [서열 확인하기] 버튼을 누르면 끝입니다!
먼저 사진을 불러와주세요!
